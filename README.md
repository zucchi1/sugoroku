# sugoroku
team-9 of ASE-class
## 仕様の概要
 - 一人での就活双六
## User Story
### 活動
- ゲーム開始
- ゲームが進む
- ゲームの終了
### ユーザーがすること＆付随するタスク
- ゲーム開始の確認
  - "Game Start"と表示
  - 盤面の決定
  - 初期化（Playerの位置）
- サイコロを振る
  - 1~3の乱数

- 位置を確認する
  - 現状の表示
    - ゴールまであと何マスか
- イベント
  - 現状決めていない
- 結果の確認
  - もし終わったら
    - 終了画面の表示
    - フィードバック
  - ゴールに達していなければ
    - サイコロを振るフェーズに
